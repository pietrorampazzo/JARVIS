# 🤖 JARVIS Agent Registry

Este documento contém o índice oficial e atualizado de todos os agentes inteligentes que operam no diretório `.agents/`.

## Agentes Ativos

### 1. Efficiency Agent (`efficiency_agent.py`)
- **Papel**: Auditor de Higiene do Workspace.
- **Função**: Identifica e remove pátios (pastas) obsoletos e scripts órfãos. Utiliza o `workspace_blueprint.md` como guia de governança.
- **Frequência**: Rodagem diária via Cron.

### 2. Cron Agent (`cron_agent.py`)
- **Papel**: Orquestrador de Agendamentos (Windows).
- **Função**: Interface central para o `schtasks` do Windows. Gerencia a persistência de rotinas no arquivo `engine/config/cron.json`.
- **Comandos**: `list`, `sync`, `add`.

### 3. Arte Heavy Agent (`arte_heavy_agent.py`)
- **Papel**: Gatilho Condicional de Matching (ARTE).
- **Função**: Monitora a divergência de dados entre `master.xlsx` e `master_heavy.xlsx`. Ativa a Engine de Matching (`arte_heavy.py`) apenas quando há pendências reais.
- **Frequência**: Verificação a cada 30 minutos.

### 4. ARTE Edital Agent (`edital_agent.py`)
- **Papel**: Orquestrador de Workflow de Editais.
- **Função**: Orquestra o workflow de editais: descompactação, localização de TR e extração Zero Token.
- **Frequência**: Monitoramento Real-time.

---

> [!IMPORTANT]
> **Protocolo de Expansão**: Sempre que um novo agente for criado neste diretório, este arquivo DEVE ser refatorado para incluir a nova estrutura e propósito do agente.
