# 📸 Fotos dos Produtos — Arte Comercial

## Padrão de Nome

```
{EDITAL}_{UASG}_{N_ITEM}_{FOTO}.jpg
```

## Exemplos

```
PE001_987654_01_1.jpg   ← Edital PE001 | UASG 987654 | Item 01 | Foto 1
PE001_987654_01_2.png   ← mesma combinação            | Item 01 | Foto 2 (png ok!)
PE001_987654_01_3.jpg   ← mesma combinação            | Item 01 | Foto 3
PE023_654321_07_1.png   ← Edital PE023 | UASG 654321 | Item 07 | Foto 1
```

## Regras
- Máximo **3 fotos por produto** (`_1`, `_2`, `_3`)
- Formatos aceitos: **`.jpg`** ou **`.png`**
- Sem espaços, sem acentos no nome do arquivo
- Resolução ideal: **800x800px** (quadrada)
- Tamanho máximo: **500KB** por arquivo
