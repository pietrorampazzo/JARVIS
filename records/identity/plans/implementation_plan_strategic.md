# Plano Estratégico: Centralização de Projetos Core

Este plano define o racional e os passos para organizar os três pilares de desenvolvimento do ecossistema Pietro/JARVIS.

## 🎯 Racional
Para que o JARVIS (Segundo Cérebro) funcione como um assistente guiado assertivo, ele precisa de **Vetor de Direção**. Monitorar apenas "atividade git" é o nível 1. O nível 2 é monitorar **Progresso rumo a Metas**.
Ao centralizar planos e metas agora, transformamos o COS em um sistema de **Governança por Objetivos (OKRs)**.

## 🏗️ Projetos Alvo
1.  **`arte_`**: A infraestrutura de licitações e business.
2.  **`wappi`**: Projeto Startup Externa (Clientes).
3.  **`JARVIS`**: O sistema de controle e governança (COS).

## 📋 Proposta de Implementação

### 1. Documentação de Identidade Ativa
Criar/Atualizar o `README.md` e `PLAN.md` de cada projeto com:
- **Visão 10x**: Onde o projeto deve estar em 6 meses.
- **Top 3 Metas**: Objetivos imediatos.
- **Dívida Técnica**: TODOs prioritários detectados pelo Sentinel.

### 2. Mapa Visual (Mission Control)
Criar um artefato `CORE_ROADMAP.md` contendo um diagrama Mermaid integrando os projetos ao core do JARVIS.

### 3. Integração ao Score Engine
Ajustar o cálculo de score para dar bônus quando ações de log estiverem vinculadas a metas de um projeto Core.

### 4. Interface Conversacional (OpenClaw Bridge) [REFINADO]
Desenvolver o bridge para que o JARVIS atue como um **Colega Autônomo**:
- **OpenClaw**: Interface única para input/output de Markdowns cerebrais.
- **Cadência de 30 min**: O JARVIS revisa tópicos e colabora ativamente, não apenas reage.
- **Pings de Parceria**: Sugestões de melhoria e refinamento de tarefas ao longo do dia.

### 10. XP BARSI & Evolution API [REFINADO]
- **CRM Trello**: Importação via Excel com template definido (#TASK).
- **Evolution API**: Monitoramento de conversas, briefing de histórico e extração de tópicos críticos (Seguros, Crédito, etc).
- **Input JSON**: IA processa conversas e gera tasks automáticas com deadline.

### 11. ARTE: Gestão de ATAs [NEW]
- **Metodologia**: Site de ATAs, Marketing via E-mail/WhatsApp.
- **Negócio**: ATAs como vertente independente de receita.

### 12. ARTE B2C [NEW]
- **E-commerce**: Criação da loginha B2C da Arte.
